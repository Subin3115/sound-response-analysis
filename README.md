# Sound Response Analysis

## Overview

How do people respond to different types of sounds?

This project investigates differences in user responses to natural and artificial sounds by analyzing YouTube comments from six sound-related videos.
The project was conducted as a team project for a Web & Text Mining course.

## Research Question

**Do people exhibit different response patterns when exposed to natural versus artificial sounds?**

## Data Collection

Comments were collected using the YouTube Data API.

### Natural Sounds

* Forest sounds
* Ocean sounds
* Valley stream sounds

### Artificial Sounds

* Subway sounds
* Library white noise
* Cafe white noise

Up to 500 comments were collected from each video.

## Text Preprocessing

The following parts of speech were extracted and analyzed:

* Adjectives
* Verbs
* Common nouns
* Proper nouns

## Analysis Methods

* Word Cloud
* Network Graph
* TF-IDF
* Cosine Similarity
* Euclidean Distance
* K-Means Clustering
* PCA
* LDA Topic Modeling
* Coherence Score Evaluation
